# 🦞 OpenClaw for Raspberry Pi 5 (The Lazy Way)

The easiest way to get OpenClaw on your Pi 5. One command, zero headache.

## 🚀 One-Click Install
Copy and paste this into your terminal:

```bash
curl -sSL https://raw.githubusercontent.com/jhawpetoss6-collab/pi-apps-openclaw/main/openclaw-pi-super-installer.sh | bash
```

## 🛠️ What happens?
- **Auto-Store**: If you have Pi-Apps, it adds OpenClaw to your GUI store automatically.
- **Auto-ARM**: Detects Pi 5 hardware and installs optimized Node.js + dependencies.
- **Auto-Update**: Sets up a background task to keep OpenClaw updated forever.
- **Auto-Start**: Creates a Desktop Icon and a Background Service.

No configuration needed. Just run and let the lobster take over.
